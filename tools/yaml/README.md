# YAML 配置文件工具

## 功能简介

提供 YAML 配置文件的加载和读取功能，包含错误处理和日志记录。

## 主要功能

- **文件加载**：从文件路径加载 YAML 配置文件
- **数据读取**：从 YAML 节点中读取指定键的值
- **错误处理**：自动处理文件不存在和解析错误，并记录日志
- **类型安全**：支持模板化的类型读取

## 使用说明

- `load(path)`：加载 YAML 文件，失败时会记录错误并退出程序
- `read<T>(yaml, key)`：从 YAML 节点中读取指定键的值，键不存在时会记录错误并退出程序
- 所有错误都会通过日志系统记录
