# Math Tools 数学工具

## 功能简介

提供常用的数学计算工具函数，包括角度处理、坐标转换、时间计算等。

## 主要功能

- **角度处理**：将弧度值限制在 (-π, π] 范围内
- **欧拉角转换**：
  - 四元数转欧拉角
  - 旋转矩阵转欧拉角
  - 欧拉角转旋转矩阵
- **坐标转换**：
  - 直角坐标系与球坐标系之间的转换（xyz ↔ ypd）
  - 提供转换函数的雅可比矩阵
- **时间计算**：计算两个时间点之间的差值（秒）
- **向量计算**：计算两个二维向量之间的夹角（0 ~ π）
- **数值限制**：限制数值在指定范围内

## 使用说明

所有函数位于 `tools` 命名空间下，可直接调用。坐标转换函数支持自定义旋转顺序（内旋/外旋）。
