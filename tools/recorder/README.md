# Recorder 记录器

## 功能简介

用于录制视频和姿态数据，支持异步保存，避免阻塞主线程。

## 主要功能

- **视频录制**：使用 OpenCV 录制视频文件
- **姿态记录**：记录每帧图像对应的四元数姿态数据
- **时间戳记录**：记录每帧的时间戳信息
- **异步保存**：使用独立线程进行文件保存，提高性能
- **帧率控制**：支持自定义录制帧率

## 使用说明

- 初始化时设置录制帧率（默认 30 fps）
- 通过 `record(img, q, timestamp)` 方法记录图像、姿态和时间戳
- 数据会被异步保存到视频文件和文本文件中
- 析构时会自动停止录制并保存所有数据
