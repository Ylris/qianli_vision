# RANSAC Sine Fitter 正弦函数拟合器

## 功能简介

基于 RANSAC（随机采样一致性）算法拟合正弦函数，用于从带噪声的数据中提取正弦信号参数。

## 主要功能

- **正弦拟合**：拟合形如 `A*sin(ω*t + φ) + C` 的正弦函数
- **RANSAC 算法**：使用 RANSAC 算法处理异常值和噪声
- **参数估计**：估计振幅 A、角频率 ω、相位 φ 和偏移 C
- **内点统计**：统计拟合结果中的内点数量

## 使用说明

- 初始化时需要设置最大迭代次数、阈值和角频率范围
- 通过 `add_data(t, v)` 添加数据点
- 调用 `fit()` 进行拟合，结果存储在 `best_result_` 中
