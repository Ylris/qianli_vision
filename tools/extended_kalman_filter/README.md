# Extended Kalman Filter 扩展卡尔曼滤波器

## 功能简介

实现扩展卡尔曼滤波器（EKF），用于非线性系统的状态估计和滤波。

## 主要功能

- **状态预测**：基于状态转移模型进行状态预测
- **状态更新**：基于观测数据更新状态估计
- **非线性支持**：支持自定义的非线性状态转移函数和观测函数
- **卡方检验**：提供 NIS（归一化创新平方）和 NEES（归一化估计误差平方）统计检验

## 使用说明

- 初始化时需要提供初始状态 `x0` 和初始协方差矩阵 `P0`
- 可自定义状态加法函数（用于处理角度等特殊状态空间）
- 支持线性或非线性的预测和更新过程
- 提供卡方检验数据用于评估滤波器性能
