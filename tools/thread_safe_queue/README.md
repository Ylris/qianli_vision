# Thread Safe Queue 线程安全队列

## 功能简介

提供线程安全的数据队列实现，支持多线程环境下的数据生产和消费。

## 主要功能

- **线程安全操作**：所有队列操作都是线程安全的
- **容量限制**：支持设置最大队列容量
- **满队列处理**：支持两种策略：
  - 队列满时调用回调函数
  - 队列满时自动弹出最旧的数据（FIFO）
- **阻塞操作**：支持阻塞式的数据获取操作

## 使用说明

- 初始化时设置最大容量和满队列处理回调函数
- 通过 `push()` 添加数据，通过 `pop()` 获取数据
- 支持阻塞和非阻塞的数据访问方式
- 可通过 `empty()` 检查队列是否为空，通过 `clear()` 清空队列
