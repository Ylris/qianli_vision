# Thread Pool 线程池

## 功能简介

提供线程池和有序队列功能，用于管理多线程任务执行和有序数据处理。

## 主要功能

- **线程池**：管理固定数量的工作线程，执行提交的任务
- **有序队列（OrderedQueue）**：保证数据按照 ID 顺序处理，即使数据乱序到达
- **YOLO 模型创建**：提供创建 YOLO11s 和 YOLOv8s 模型的辅助函数
- **Frame 结构**：定义包含图像、时间戳、姿态和检测结果的帧数据结构

## 使用说明

- **ThreadPool**：通过 `enqueue()` 方法提交任务，线程池会自动分配线程执行
- **OrderedQueue**：通过 `enqueue()` 添加帧数据，通过 `dequeue()` 按顺序获取数据
- 支持阻塞和非阻塞的数据获取方式
