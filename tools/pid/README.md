# PID 控制器

## 功能简介

实现经典的 PID（比例-积分-微分）控制器，用于闭环控制系统。

## 主要功能

- **PID 控制**：基于目标值和反馈值计算控制输出
- **参数可配置**：支持设置 P、I、D 系数
- **输出限制**：支持总输出和积分项输出的最大值限制
- **角度模式**：支持角度控制模式（自动处理角度周期性）
- **调试支持**：提供 P、I、D 各项的输出值用于调试

## 使用说明

- 初始化时需要设置控制周期 `dt`、PID 系数（kp、ki、kd）和输出限制
- 通过 `calc(set, fdb)` 方法计算控制输出，其中 `set` 为目标值，`fdb` 为反馈值
- 可通过 `pout`、`iout`、`dout` 成员变量查看各项输出
