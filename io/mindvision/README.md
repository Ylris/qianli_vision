# MindVision 模块

迈德威视相机驱动模块，用于控制迈德威视工业相机。

## 功能

- 基于迈德威视SDK（CameraApi）实现相机控制
- 支持通过VID/PID识别和选择特定相机设备
- 可配置曝光时间（毫秒）和伽马值参数
- 使用独立线程进行图像采集
- 支持USB设备重置功能，提高设备稳定性
- 实现CameraBase接口，可被Camera工厂类统一管理

## 主要类

- `MindVision`: 迈德威视相机类，继承自 `CameraBase`
  - `read()`: 读取图像和时间戳

## 特性

- 多线程架构：独立的采集线程和守护线程
- 自动设备管理：支持USB设备重置和重连
- 参数可配置：曝光时间、伽马值等参数可在初始化时设置
