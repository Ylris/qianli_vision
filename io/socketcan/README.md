# SocketCAN 模块

SocketCAN通信模块，用于Linux系统下的CAN总线通信。

## 功能

- 基于Linux SocketCAN接口实现CAN总线通信
- 支持CAN帧的发送和接收
- 使用epoll实现高效的异步I/O
- 自动重连机制，提高通信稳定性
- 支持自定义接收回调函数处理CAN帧

## 主要类

- `SocketCAN`: CAN总线通信类
  - `write()`: 发送CAN帧
  - 接收通过回调函数处理

## 特性

- 多线程架构：独立的接收线程和守护线程
- 自动重连：当连接断开时自动尝试重连
- 高效I/O：使用epoll实现非阻塞I/O操作
- 异常处理：完善的错误处理和日志记录

## 使用场景

- 控制板通信（cboard模块）
- 其他需要CAN总线通信的设备
