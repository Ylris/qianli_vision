# USBCamera 模块

USB相机驱动模块，用于控制标准USB接口的相机设备。

## 功能

- 基于OpenCV VideoCapture实现USB相机控制
- 支持相机参数配置：曝光、帧率、锐度、伽马值、增益等
- 使用独立线程进行图像采集，提高性能
- 自动重连机制，确保设备稳定性
- 支持图像尺寸配置

## 主要类

- `USBCamera`: USB相机类
  - `read()`: 读取图像或图像+时间戳

## 特性

- 多线程架构：独立的采集线程和守护线程
- 自动重连：当设备断开时自动尝试重连
- 参数可配置：曝光、帧率、锐度等多种参数可在配置文件中设置
- 线程安全：使用互斥锁保护相机操作

## 使用场景

- 标准USB接口的相机设备
- 需要快速部署和测试的场景
