# CBoard 模块

控制板通信模块，用于与主控板进行CAN总线通信。

## 功能

- 通过SocketCAN与主控板进行CAN总线通信
- 接收IMU四元数数据，支持时间戳查询
- 发送控制命令（包括控制模式、射击指令、yaw/pitch角度等）
- 接收子弹速度、工作模式等状态信息
- 支持多种工作模式：空闲、自瞄、小符、大符、前哨站
- 支持哨兵专用射击模式：左射、右射、双射

## 主要类

- `CBoard`: 控制板通信类
  - `imu_at()`: 根据时间戳获取IMU四元数数据
  - `send()`: 发送控制命令到主控板

## 通信协议

- 使用CAN总线进行通信
- 支持接收IMU四元数、子弹速度等数据
- 支持发送云台控制命令
