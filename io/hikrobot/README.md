# HikRobot 模块

海康机器人相机驱动模块，用于控制海康机器人工业相机。

## 功能

- 基于海康机器人SDK（MvCameraControl）实现相机控制
- 支持通过VID/PID识别和选择特定相机设备
- 可配置曝光时间（微秒）和增益参数
- 使用独立线程进行图像采集，提高性能
- 支持USB设备重置功能，提高设备稳定性
- 实现CameraBase接口，可被Camera工厂类统一管理

## 主要类

- `HikRobot`: 海康机器人相机类，继承自 `CameraBase`
  - `read()`: 读取图像和时间戳

## 特性

- 多线程架构：独立的采集线程和守护线程
- 自动设备管理：支持USB设备重置和重连
- 参数可配置：曝光时间、增益等参数可在初始化时设置
